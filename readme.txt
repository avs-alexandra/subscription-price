Manage subscriptions in WooCommerce by changing user roles and sending notifications.

== Description ==
**Subscription Price** — это плагин для WordPress, который расширяет функционал WooCommerce для управления подписками. С помощью этого плагина вы можете:
- Назначать пользователям роли в зависимости от их статуса подписки.
- Автоматически уведомлять пользователей о скором завершении или истечении подписки.
- Использовать шорткоды для отображения информации о подписке.
- Управлять настройками подписок, уведомлений и ролей пользователей через админку WordPress.

**Key Features:**
- **Управление ролями:** Назначение активной и истёкшей роли в зависимости от статуса подписки.
- **Уведомления:** Автоматическая отправка писем с напоминаниями и уведомлениями об истечении подписки.
- **Шорткоды:** Используйте `[subscription_status]` для отображения даты окончания подписки для авторизованных пользователей.
- **Интеграция с профилем пользователя:** Добавление информации о подписке в профиль пользователя в админке.
- **Настраиваемые параметры:** Конфигурация шаблонов писем, длительности подписки и других опций.

== Installation ==
1. Скачайте плагин.
2. Загрузите папку с плагином в директорию `/wp-content/plugins/`.
3. Активируйте плагин через меню "Плагины" в админке WordPress.

== Frequently Asked Questions ==
**В: Требуется ли WooCommerce для работы плагина?**  
О: Да, WooCommerce должен быть установлен и активирован.

**В: Как настроить шаблоны писем?**  
О: Перейдите в **WooCommerce > Маркетинг > Настройки Подписки** и отредактируйте темы и текст писем в разделе «Настройки писем».

**В: Можно ли управлять несколькими планами подписок?**  
О: Да, плагин поддерживает до 4 планов подписки с разными продуктами, длительностью и ролями.

**В: Что происходит при истечении подписки?**  
О: Плагин автоматически меняет роль пользователя на «истёкшую» и отправляет уведомление об истечении.

= Можно ли использовать плагин без WooCommerce? =
Нет, плагин предназначен только для сайтов WooCommerce.
